# HackWare CyberFunk

Juego arcade multijugador de programación hecho con Node.js, Express y Socket.IO.

## Requisitos

- Node.js 18+
- npm

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm start
```

La aplicación estará disponible en http://localhost:3000.

## Estructura

- `server/`: lógica del servidor, salas y minijuegos
- `public/`: frontend estático y assets
